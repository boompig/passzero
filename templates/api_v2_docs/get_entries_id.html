GET /api/v2/entries/<int:entry_id>

Decrypt the given entry and return the contents

Arguments

None

Response

on success: 


entry

exactly what information is returned depends on the entry version on error: 


{ "status": "error", "msg": string }

Status codes

  • 200 - success
  • 500 - there are some old entries (version < 4) so this method cannot work
