POST /api/v1/user/recover

First step of account recovery for the specified user. Send an account recovery token to the user's email address.

Arguments 


{
	"csrf_token": string (required),
	"email": string (required)
}

Response 


{ "status": "success"|"error", "msg": string }

Status codes

  • 200 - success
  • 400 - failed to validated parameters
  • 401 - no account for this email
  • 500 - internal server error, or email failed to send
