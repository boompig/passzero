GET /api/v1/docs

Retrieve all saved documents for the logged-in user The document contents will be encrypted.

Arguments

None

Response

on success: 

[doc-1, doc-2, ... doc-n]

a document's object looks like this: 

{
"id": number,
"name": string,
"contents": binary
}

The contents will be encrypted on error: 

{ "status": "error", "msg": string }

Status codes

  • 200 - success
  • 401 - user is not logged in
