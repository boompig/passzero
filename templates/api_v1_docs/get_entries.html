GET /api/v1/entries

Retrieve all decrypted entries for the logged-in user

Arguments

None

Response

on success: 

[entry-1, entry-2, ... entry-n]

The entry details depend on which version entries the user has. For details see passzero/models.py. On error: 

{ "status": "error", "msg": string }

Status codes

  • 200 - success
  • 401 - user is not logged in
