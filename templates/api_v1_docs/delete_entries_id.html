DELETE /api/v1/entries/<int:entry_id>

Delete the entry with the given ID.

Arguments


{
    "csrf_token": string (required)
}

Response

Success: 

{
    "status": "success",
    "msg": string
}
Error: 
 
{
    "status": "error",
    "msg": string
}

Status codes

  • 200 - success
  • 400 - entry does not exist or does not belong to logged-in user
  • 401 - not authenticated
  • 403 - CSRF check failed
