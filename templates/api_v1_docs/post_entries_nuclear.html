POST /api/v1/entries/nuclear

Delete all entries for the logged-in user.

Arguments


{
    "csrf_token": string (required)
}

Response

Success: 

{
    "status": "success",
    "msg": string
}
Error: 
 
{
    "status": "error",
    "msg": string
}

Status codes

  • 200 - success
  • 401 - not authenticated
  • 403 - CSRF token validation failed
